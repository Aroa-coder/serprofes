## ▶️ Cómo ejecutar los ejercicios

### HTML / CSS / JavaScript

Abrir el archivo `index.html` en el navegador.

### Node.js

Instalar dependencias:

```bash
npm install
```

Ejecutar un ejercicio:

```bash
node archivo.js
```

O si existe:

```bash
npm start
```
